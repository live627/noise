# noise
Header-only Perlin noise library for modern C++

Based on Ken Perlin's [Improved Noise](<http://mrl.nyu.edu/~perlin/noise/>).

## Features
* random seed
* 1D/2D/3D noise
* octave noise
* [0.0, 1.0] noise
